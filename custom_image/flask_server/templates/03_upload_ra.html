제출 방법

  1. 영수증 파일(엑셀)을 제출한다.
  2. 회의록 파일(한글)을 제출한다.
  3. 회의록 파일(pdf)을 제출한다.

주의사항

  1. 다운받은 영수증 파일의 이름을 수정하지 않는다.
  2. 영수증 파일의 형식을 변경하지 않는다.
  3. 영수증이 너무 많은 경우, 페이지 레이아웃을 확인하고 2페이지에 추가한다.
  4. 회의록 파일의 제목은 영수증 파일의 제목을 복사 붙여넣기 후, (영)을 (회)로 변경한다.
  5. 회의록 파일 pdf는 한글 파일명과 동일하게 설정한다.
