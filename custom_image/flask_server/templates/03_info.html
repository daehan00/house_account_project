양식파일 다운로드

1. 카드 사용 전:

  • 미리 언제, 얼마를 결제할 지 준비한 후 달력 페이지에 예약하세요.
  • 예약이 겹치는 경우 먼저 예약한 사람이 우선입니다.
  • 필요하다면 서로 조정하면 됩니다.

2. 영수증 및 회의록 제출:

  • 카드를 사용했다면 사용내역을 제출해야 합니다.
  • 제출해야 하는 항목은 다음과 같습니다: 영수증내용, 영수증엑셀파일, 회의록, (기념품지급대장 등)
  • 매월 1-15일 사이 결제건 => 15일까지
  • 15일 이후 결제건 => 말일까지

3. 실물영수증 제출:

  • 영수증 엑셀파일을 인쇄하고 실물영수증을 부착해서 제출해야 합니다.
  • 매월 말일까지 제출하세요.

사용 가능시간:

  • 평일 오전 8:00 - 오후 11:00
  • 외 시간에 사용 필요시 회계 RA에게 문의하세요.

분할결제:

  • 분할결제는 절대 금지입니다.
  • 분할결제란? : 같은 항목을 여러번에 나눠서 결제하는 것
  • 최대한 사야 할 물건은 한번에 결제하세요.

결제금액:

  • 온라인 결제 시 30만원 초과하면 회계 RA에게 보고하기!
  • 현장결제(카드로 결제) 시 제한 없습니다.
  • 식비 : 인당 2만원 한도, 간식비: 인당 1만원 한도
  • 식비 + 간식비 : 인당 2만원 한도
  • 한도 피하기 위해 분할결제 절대 금지!
  • 프로그램 내 집행금액 총합 50만원 넘을 시 회계 RA에게 문의하세요.

결제를 잘못한 경우:

  • 주소를 잘못 입력하거나, 결제해야 할 물품을 빠트렸거나 하는 이유로 결제를 취소해야 하는 경우에는 어떻게 해야할까요?
  • 결제를 잘못 한 경우에는 당일에 취소해야 합니다.
  • 나중에 알게 되어서 취소해야 하는 경우, 회계 RA에게 문의하세요.

결제 불가 품목입니다.

  • 이거 사도 되나? 싶은 것들은 회계 RA에게 질문하세요.
  • 술!
  • 교통비
  • 입장료
  • 상품권
  • 송도, 신촌 외 현장결제
